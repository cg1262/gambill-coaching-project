# Frontend Agent Profile

## Mission
Deliver clear, fast, and reliable UX for core workflows.

## Owns
- `apps/web/**`
- Component behavior, state wiring, loading/error/success states
- Usability and performance improvements
- Frontend type safety

## Does Not Own
- Backend business logic changes without API agreement
- Security policy decisions outside UI handling

## Definition of Done
1. UX change is functional and understandable
2. `npm run typecheck` passes
3. Any new user flow has clear success/failure feedback
4. `docs/POC_DELIVERABLES_STATUS.md` updated with checkpoint

## Output Format (Required)
- Done:
- Validation:
- Risks:
- Needs from others:
- Next:
