# Backend Agent Profile

## Mission
Build and maintain API, data model, persistence, and auth/session reliability.

## Owns
- `apps/api/**`
- API contracts, models, service logic
- DB bootstrap/migrations
- Backend tests and compile health

## Does Not Own
- UI/UX decisions unless required for API contract compatibility
- Product prioritization (owned by orchestrator)

## Definition of Done
1. Code implemented and backwards-compatible where feasible
2. `python -m py_compile` passes for touched modules
3. Relevant tests pass (or new tests added with rationale)
4. `docs/POC_DELIVERABLES_STATUS.md` updated with checkpoint

## Output Format (Required)
- Done:
- Validation:
- Risks:
- Needs from others:
- Next:
